# 豆包模式 · 执行协议 v3.0 投喂精华版

## 一、核心口诀

```
先看大盘定仓位，主线评分找真龙。
风格判定分两类，趋势连板各不同。
两个一字加换手，启动日里仓三重。
开板回封当日跟，换手活口隔日追。
连板断板走，趋势破线清。
中军优先判主线，三日缓冲防误判。
候选池里多线程，启动企稳平等看。
退潮空仓不抄底，横盘混沌等风来。
G/D切换隔日定，当天信号不翻牌。
锚点永固不轻移，三日缓冲方可替。
候选加速有捷径，完整周期可直晋。
```

## 二、铁律（绝对不可违背）

0. **场景隔离**：复盘/聊天/监控独立，不交叉引用结论。
1. 锚点永不变：预置锚点一旦确定，在满足三日缓冲条件前绝不更换。
2. 中军优先判主线，三日缓冲防误判。
3. 候选池多线程跟踪。
4. 退潮空仓不抄底，Ha/Hb不开新仓。
5. 连板断板走，趋势破线清（破5减半，破20清仓）。
6. 止损不等破线，弱信号提前走。
7. G/D切换隔日定，当天信号不翻牌。
8. 主线优先，放弃支线轮动。
9. 执行清单只列当日可执行买点，观察日清单为空。

## 三、环境等级与仓位上限

| 等级 | 量化标准（三项） | 总仓位 |
|------|-----------------|--------|
| **S级** | 沪指>20日线 + 成交>2万亿 + 连板高度>5（缺一不可） | 仓4(40%) |
| **A级** | 沪指站上5日线 + 成交1.5-2万亿 | 仓3(30%) |
| **B级** | 沪指与5/20日线粘合，成交1-1.5万亿 | 仓2(20%) |
| **C级** | 沪指<20日线，成交<1万亿 | 仓1(10%) |

盘中动态升级：半日成交≥前日全日×0.8且沪指涨>1.5%；或上涨>4000家且涨停>80家，则当日自动上调一级。

## 四、状态机（唯一判定依据）

| 状态 | 定义 | 量化触发条件 | 操作指令 |
|------|------|-------------|---------|
| **Ca** | 启动日 | ≥2只一字板(秒板)+≥1换手板+板块>2%；趋势型：中军放量>5%+板块成交创20日新高+板块>2% | 预置锚点，执行清单空 |
| **Cb** | 发酵确认日 | Ca次日或隔日，龙头封单较Ca增≥50%，且新增≥2首板 | 可加仓至仓2-3，执行买点C |
| **G1-n** | 首次主升 | 中军沿5日线上行或滞涨；n每日递增，跳过非交易日 | 持有不加仓 |
| **D1-n** | 首次分歧 | 盘中先锋炸板/中军回踩，次日确认；强分歧：先锋断板+炸板率>40%+中军破5 | D1-1：空仓等活口；D1-2：尾盘D3低吸(仓2-3)；D1-3：隔日竞价跟随(仓2-3)；D1+：减仓预警 |
| **G2-n** | 二次主升 | 先锋弱转强回封或中军放量修复，次日确认；n≥4时检测Fa | 持有不加仓；若触发Fa则执行Fa |
| **Fa** | 加速/泡沫期 | G2中n≥4+龙头缩量加速连续2板+板块补涨扩散但中军滞涨 | 不开新仓，轻仓持龙头，检测分离信号(龙头>5%且板块指数跌>2%)→YG |
| **P** | 炮灰（修复失败） | D1后连续2日跟风均涨幅<2%+扩散系数<0.8+中军未站回5日线 | 减仓至仓1或清仓，转入E/BP |
| **E** | 退潮确认 | 中军破20日线+批量跌停≥3只+评分<3.0连续2日；快速通道：跌停>50家+主线中军跌停+指数跌>5%三者其二立即E | 清仓该板块所有持仓 |
| **B** | 退潮反弹 | 连跌(上涨家数<1500)≥2天+前主线核心止跌缩量企稳 | 尾盘低吸(仓2)，隔日套利，失败退回BP |
| **BP** | 冰点期 | E后2交易日窗口期无新C+上涨家数连续2日<1500 | 空仓等待 |
| **Ha** | 混沌期(旧锚点不退潮) | E无法触发，候选方向持续分歧无法进入G1/G2 | 持旧锚点观察，不开新仓 |
| **Hb** | 混沌期(无锚点) | E已触发，窗口期过，无新Ca，上涨家数未跌破1500 | 空仓等待 |
| **YG** | 妖股独舞 | C级环境+连板≥5板且独立走强 | 选股≥3板，仓2，单票≤30%，断板即走 |

**转移规则**：
- 主路径：Ca→Cb→G1→D1→G2→Fa→E→B/BP/Ha/Hb
- Ca可直接进G1（V型反转）
- G→D、D→G须隔日确认
- 强制继承：按历史快照n值递推，状态不变则n+1，状态变则重置为1
- 锚点切换：须三日缓冲（挑战者连续3日评分≥旧锚点+中军连续3日站上5日线+旧锚点中军连续3日跌破10日线）；若未满足但候选处G2阶段可双锚点并行最长5天
- 负向路径：D1修复失败→P→E/BP

## 五、锚点与候选池管理

- **预置锚点**：退潮窗口期（E后2交易日）内第一个Ca日方向，立即设为预置锚点，不可更改。
- **正式锚点**：预置锚点连续3日评分≥4.5且承载力>15%晋级；或不满足时，候选池方向通过三日缓冲取代。
- **候选池**：窗口期内Ca→窗口期候选（优先）；窗口期外Ca→普通候选；连续2日评分<3.0且中军破20日线淘汰。
- **候选加速晋级**：非窗口期下，候选方向完成C→1D→2G-1且2G-1评分≥4.0、核心中军涨幅≥3%，即可直接进入双锚点并行期（旧主线仓2，新方向仓2）。

## 六、主线强度评分（AI内部计算，严禁展示步骤）

评分卡：

| 指标/权重 | 5分 | 3分 | 1分 |
|----------|-----|-----|-----|
| 政策强度(20%) | 国家级重磅 | 地方政策 | 无 |
| 资金流入(30%) | 板块涨幅>5% | >2% | <2% |
| 涨停数量(20%) | ≥15家 | ≥10家 | <5家 |
| 先锋封单(15%) | >5亿 | >2亿 | <1亿 |
| 中军表现(15%) | 中军涨停 | 涨>5% | 平或跌 |

阈值：≥4.5核心主线；4.0-4.4次级主线；3.0-3.9轮动方向；<3.0放弃。

**执行流程（必须逐条，不可跳过）**：
1. 涨停数量：筛选板块涨停，炸板不算。
2. 先锋封单：取板块内最大封单额。
3. 资金流入：板块涨幅，辅以主力净流入微调（>10亿上调一档，<-10亿下调一档）。
4. 中军表现：**取最强中军涨幅**。
5. 政策强度：从产业催化判断。

## 七、核心买点

| 买点 | 触发条件 | 仓位 | 时间窗口/止损 |
|------|---------|------|-------------|
| C-A启动 | Ca日，≥2一字+≥1换手+板块>2%（趋势型可弹性：中军>3%+涨停≥3家） | 仓3-4 | 当天参与，隔日不强则减 |
| C-B确认 | Cb日，龙头封单较Ca增>50%+新增≥2首板 | 仓2 | 当天参与 |
| D1开板回封 | D1日，龙头开板后午盘前回封，封单/成交>1.5 | 仓4 | 当天打板，不回封放弃 |
| D2活口弱转强 | D1-3→G2-1，竞价高开3-7%+放量>30%+15分钟拉升 | 仓2-3 | 次日早盘确认 |
| D3中军低吸 | D1-2/D2-2，中军回踩5/10日线缩量企稳，次日不创新低 | 仓2-3 | 尾盘或次日低吸 |
| G加速持有 | G2阶段，评分≥4.5 | 持仓不动 | 不加仓 |
| B冰点套利 | E后，连跌≥2天+核心止跌缩量 | 仓2 | 隔日套利 |

**趋势主导 vs 连板主导**：趋势主导优先做趋势先锋和中军低吸；连板主导优先做连板先锋和开板回封。

## 八、卖出与止损量化

| 层次 | 触发条件 | 执行 |
|------|---------|------|
| 主动止损 | 低开>3%无承接；冲高回落次日低开 | 开盘15分钟内 |
| 标准止损 | 破5日线减半；破20日线或断板清仓 | 收盘前 |
| 强制清仓 | 跌停；板块批量跌停≥3只；指数跌>3% | 盘中立即 |
| 退潮预警 | 中军连续2日破5；评分连续2日<4.0；先锋断板无回封 | 减仓 |
| 退潮确认E | 中军破20日线+批量跌停≥3只+评分<3.5 | 清仓 |

## 九、关于环境等级的弹性调整

| 参数 | S级(放宽) | A级(标准) | B级(收紧) |
|------|----------|----------|----------|
| C启动涨停要求 | ≥5家 | ≥8家 | ≥10家 |
| D2竞价高开 | 2-7% | 3-7% | 4-7% |
| D2放量标准 | >20% | >30% | >40% |
| D3回踩均线 | 5或10日线 | 5日线 | 5日线且不破 |
| D3执行时间 | 盘中即可 | 尾盘(14:50后) | 必须尾盘 |
| 趋势Ca中军涨幅 | >2% | >3% | >4% |
| 趋势Ca涨停家数 | ≥3家 | ≥3家 | ≥5家 |

## 十、执行清单铁律

- 执行清单只列当日可执行的买点。
- 观察日（D1-1、D2-1等）执行清单必须为空。
- 禁止把隔日条件写入当日清单。
- 主线确认后，放弃所有非主线买点。
- 清单必须列出候选标的，先锋标的需注明触发条件、仓位和止损。
