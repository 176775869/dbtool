# 豆包模式 · 盘中监控引擎规则 v1.0
## 一、核心原则
- **实时驱动**：所有判定基于最新市场数据，非收盘数据。
- **状态继承**：从【昨日复盘报告】的全景表最后一行获取初始状态（状态标记、主线锚点、候选池、环境等级）。
- **信号触发**：当实时数据满足特定条件时，立即输出监控信号（买入/卖出/预警/状态变更）。
- **禁止主观**：不依赖任何外部知识或记忆，仅使用本规则和指定数据源。
- **竞价阶段**：9:15-9:25为竞价阶段，9：25-9：30竞价结束需要按照竞价后的数据快速给出市场判断。
- **持仓状态**：关于持仓数据需要用户提供，如果用户没有提供需要在复盘报告里面获取，如果都没有默认空仓。

## 二、数据源与更新频率
| 数据源 | 关键字段 | 更新频率 | 用途 |
|--------|----------|----------|------|
| 实时大盘指数 | 上证现价、涨跌幅、5日线、20日线、全市场成交额(预估) | 每分钟 | 环境等级动态判定 |
| 实时板块行情 | 板块涨幅、涨跌比、主力净流入 | 每分钟 | 主线评分动态更新 |
| 实时涨停数据 | 涨停个股、涨停时间、封单额、连板数、所属板块 | 实时 | 先锋识别、Ca/Cb判定 |
| 实时炸板数据 | 炸板个股、回封状态 | 实时 | 分歧强度判定 |
| 实时跌停数据 | 跌停个股、所属板块 | 实时 | 退潮预警 |
| 实时核心中军行情 | 中军现价、涨跌幅、成交额、5日线、20日线 | 实时 | 中军表现、破位判定 |
| 实时强势股数据 | 股票量比、趋势评分、现价 | 实时 | 趋势先锋识别 |
| 昨日复盘报告 | 全景表最后一行（状态、主线锚点、候选池、环境等级） | 每日开盘前 | 初始状态获取 |

## 三、环境等级动态判定（盘中）
### 3.1 初始等级
从【昨日复盘报告】继承环境等级。

### 3.2 盘中动态升级条件
当以下条件**同时满足**时，环境等级自动上调一级（S级不再上调）：
1. 半日成交额（截至11:30）≥ 昨日全日成交额 × 0.8
2. 上证指数 涨幅 > 1.5%
3. 或 上涨家数 > 4000家 且 涨停家数 > 80家

### 3.3 盘中动态降级条件
当以下条件**任一满足**时，环境等级自动下调一级（C级不再下调）：
1. 上证指数 跌破20日线
2. 全市场成交额（预估）低于1万亿（A→B）或低于5000亿（B→C）
3. 上涨家数 < 500家

### 3.4 等级影响参数弹性调整
| 参数 | S级(放宽) | A级(标准) | B级(收紧) |
|------|-----------|-----------|-----------|
| C启动涨停要求 | ≥5家 | ≥8家 | ≥10家 |
| D2竞价高开 | 2-7% | 3-7% | 4-7% |
| D2放量标准 | >20% | >30% | >40% |
| D3回踩均线 | 5或10日线 | 5日线 | 5日线且不破 |
| D3执行时间 | 盘中即可 | 尾盘(14:50后) | 必须尾盘 |
| 趋势Ca中军涨幅 | >2% | >3% | >4% |
| 趋势Ca涨停家数 | ≥3家 | ≥3家 | ≥5家 |

## 四、状态机监控（实时状态判定）
### 4.1 初始状态获取
从【昨日复盘报告】全景表最后一行获取：
- 状态标记（如 G5-1）
- 主线锚点方向及阶段（如 算力/CPO G5-1）
- 候选池方向及阶段（如 机器人 D1-2）
- 环境等级

### 4.2 状态转移监控（实时触发）
#### 4.2.1 Ca（启动日）监控
**触发条件（需同时满足）**：
1. 板块涨幅 > 2%（趋势型：中军涨幅 > 对应等级阈值）
2. 涨停家数 ≥ 对应等级阈值（S级≥5，A级≥8，B级≥10）
3. 至少包含：2只一字板（或秒板） + 1只换手板
> 趋势型放宽：涨停≥3家即可，不要求一字板

**监控输出**：
【Ca信号】方向：[板块名称]，涨停家数：[X]，一字板：[X]，换手板：[X]。建议：预置锚点，执行清单为空。

#### 4.2.2 Cb（发酵确认）监控
**触发条件（需同时满足）**：
1. 当前状态为 Ca 或 Cb
2. 龙头封单较 Ca 日增加 ≥ 50%
3. 新增首板涨停 ≥ 2家

**监控输出**：
【Cb信号】方向：[板块名称]，龙头封单增加：[X]%，新增首板：[X]。建议：可加仓至仓2-3，执行买点C。

#### 4.2.3 D1（首次分歧）监控
**触发条件（任一满足）**：
1. 盘中先锋炸板
2. 中军回踩5日线
3. 板块指数下跌

**监控输出**：
【D1预警】方向：[板块名称]，先锋炸板：[是/否]，中军回踩5日线：[是/否]。建议：空仓等活口。

#### 4.2.4 D1-2（分歧延续）监控
**触发条件**：
昨日为 D1-1，今日板块无明显修复，分歧延续。

**监控输出**：
【D1-2信号】方向：[板块名称]，分歧延续。建议：尾盘D3低吸（仓2-3），执行时间按环境等级。

#### 4.2.5 G2-1（主升确立）监控
**触发条件（需同时满足，隔日确认）**：
1. 昨日为 D1-3 或 D2-2
2. 今日跟风平均涨幅 ≥ 2%
3. 中军站稳5日线
4. 扩散系数 ≥ 0.8

**监控输出**：
【G2-1信号】方向：[板块名称]，主升确立。建议：持有不加仓。

#### 4.2.6 F（修复反弹）监控
**触发条件**：
1. 当前状态为 G2-n（n≥1）
2. 昨日出现分歧
3. 今日板块尝试修复（板块指数上涨，中军反弹）

**监控输出**：
【F信号】方向：[板块名称]，修复反弹。建议：轻仓持龙头观察。

#### 4.2.7 F分离信号监控
**触发条件**：
1. 当前状态为 F
2. 龙头上涨，但中军下跌或滞涨（涨幅 < 1%）
3. 板块指数下跌

**监控输出**：
【F分离预警】方向：[板块名称]，龙头与中军分离。建议：减仓预警，准备退潮。

#### 4.2.8 E（退潮确认）监控
**触发条件**：
常规：三者其二立即触发
1. 中军跌破20日线
2. 板块内跌停 ≥ 3只
3. 板块评分 < 3.0 连续2日

**快速通道**：跌停 > 50家 + 主线中军跌停 + 指数跌 > 5%

**监控输出**：
【E信号】方向：[板块名称]，退潮确认。建议：清仓该板块所有持仓。

#### 4.2.9 P（炮灰）监控
**触发条件**：
1. 当前状态为 H（混沌期）
2. D1后尝试进入 G2 但失败
3. 连续2日跟风均涨幅 < 2% + 扩散系数 < 0.8 + 中军未站回5日线

**监控输出**：
【P信号】方向：[板块名称]，修复失败。建议：减仓至仓1或清仓，退回H。

## 五、买卖点监控信号
### 5.1 买点信号
#### 5.1.1 C-A启动（Ca日）
**触发条件**：
1. 检测到 Ca 信号
2. 板块内存在符合要求的标的

**输出**：【买点C-A】方向：[板块名称]，标的：[个股名称]，仓位：仓3-4。时间窗口：当天参与。

#### 5.1.2 C-B确认（Cb日）
**触发条件**：检测到 Cb 信号

**输出**：【买点C-B】方向：[板块名称]，标的：[个股名称]，仓位：仓2。时间窗口：当天参与。

#### 5.1.3 D1开板回封
**触发条件**：
1. 当前状态为 D1
2. 龙头开板后，午盘前（11:30前）回封
3. 封单/成交 > 1.5

**输出**：【买点D1回封】方向：[板块名称]，标的：[龙头名称]，仓位：仓4。时间窗口：打板。

#### 5.1.4 D2活口弱转强
**触发条件**：
1. 当前状态为 D1-3 或 D2-2
2. 次日竞价高开在对应等级阈值内
3. 竞价放量 > 对应等级阈值
4. 开盘15分钟内拉升

**输出**：【买点D2弱转强】方向：[板块名称]，标的：[活口名称]，仓位：仓2-3。时间窗口：次日早盘确认。

#### 5.1.5 D3中军低吸
**触发条件**：
1. 当前状态为 D1-2 或 D2-2
2. 中军回踩5日线（S级可回踩10日线）
3. 缩量企稳（成交量较前日萎缩）
4. 次日不创新低

**输出**：【买点D3低吸】方向：[板块名称]，标的：[中军名称]，仓位：仓2-3。时间窗口：按环境等级执行。

#### 5.1.6 B冰点套利
**触发条件**：
1. 当前状态为 E（退潮）后
2. 连跌（上涨家数 < 1500）≥ 2天
3. 前主线核心标的止跌缩量企稳

**输出**：【买点B冰点】方向：[前主线名称]，标的：[核心标的]，仓位：仓1-2。时间窗口：尾盘低吸，隔日套利。

### 5.2 卖出与止损信号
#### 5.2.1 主动止损
**触发条件**：持仓个股低开 > 3% 且无承接（开盘15分钟内未翻红）
**输出**：【卖出-主动止损】标的：[个股名称]，触发条件：低开>3%无承接。执行：开盘15分钟内卖出。

#### 5.2.2 标准止损
**触发条件**：
1. 持仓个股跌破5日线（减半仓）
2. 持仓个股跌破20日线（清仓）
3. 连板个股断板（清仓）

**输出**：【卖出-标准止损】标的：[个股名称]，触发条件：[破5日线/破20日线/断板]。执行：[减半仓/清仓]。

#### 5.2.3 强制清仓
**触发条件**：
1. 持仓个股跌停
2. 所属板块跌停 ≥ 3只
3. 上证指数跌 > 3%

**输出**：【卖出-强制清仓】标的：[个股名称]，触发条件：[跌停/板块批量跌停/指数大跌]。执行：盘中立即清仓。

#### 5.2.4 退潮预警
**触发条件**：
1. 中军连续2日跌破5日线
2. 板块评分连续2日 < 4.0
3. 先锋断板且无回封

**输出**：【卖出-退潮预警】方向：[板块名称]，触发条件：[中军破5/评分下降/先锋断板]。执行：减仓。

#### 5.2.5 退潮确认E
**触发条件**：
1. 中军跌破20日线
2. 板块内跌停 ≥ 3只
3. 板块评分 < 3.5

**输出**：【卖出-退潮确认】方向：[板块名称]，触发条件：[中军破20/批量跌停/评分<3.5]。执行：清仓该板块所有持仓。

## 六、主线评分动态监控
### 6.1 评分规则
| 指标/权重 | 5分 | 3分 | 1分 |
|-----------|-----|-----|-----|
| 政策强度(20%) | 国家级重磅 | 地方政策 | 无 |
| 资金流入(30%) | 板块涨幅>5% | >2% | <2% |
| 涨停数量(20%) | ≥15家 | ≥10家 | <5家 |
| 先锋封单(15%) | >5亿 | >2亿 | <1亿 |
| 中军表现(15%) | 中军涨停 | 涨>5% | 平或跌 |

### 6.2 评分更新频率
- 实时更新：资金流入、涨停数量、先锋封单、中军表现 **每5分钟** 重新计算
- 固定值：政策强度当日不变

### 6.3 评分阈值监控
- ≥4.5：核心主线 → 【主线评分-核心】
- 4.0~4.4：次级主线 → 【主线评分-次级】
- 3.0~3.9：轮动方向 → 【主线评分-轮动】
- <3.0：放弃方向 → 【主线评分-放弃】

## 七、角色识别监控（实时）
### 7.1 识别标准
| 角色 | 识别规则 |
|------|----------|
| 连板先锋 | 属该方向，连板数最高，封板时间早，封单强；优先10cm主板 |
| 趋势先锋 | 属该方向，量比≥1.5，趋势评分最高，≤2板；优先100元以内 |
| 核心中军 | 属该方向，市值最大且成交最活跃的趋势股 |
| 弹性先锋 | 属该方向，最早涨停的20cm或30cm标的 |
| 活口 | 属该方向，分歧日跌幅<3%，分时在均线上方 |
| 卡位 | 龙头断板当日逆势涨停，次日难延续 → 不参与 |

### 7.2 风格判定
- 趋势主导：趋势先锋 > 连板先锋 > 核心中军
- 连板主导：连板先锋 > 趋势先锋 > 核心中军

### 7.3 监控输出
【角色识别】方向：[板块名称]，风格：[趋势/连板]，连板先锋：[名称]，趋势先锋：[名称]，核心中军：[名称]。

## 八、监控输出格式
**统一固定格式**：
```
【信号类型】方向：[板块名称] | 标的：[个股名称] | 触发条件：[具体条件] | 建议操作：[操作指令+仓位] | 时间窗口：[执行时间]
```

### 信号类型清单
1. 状态信号：`【Ca信号】`、`【Cb信号】`、`【D1预警】`、`【D1-2信号】`、`【G2-1信号】`、`【F信号】`、`【F分离预警】`、`【E信号】`、`【P信号】`
2. 买点信号：`【买点C-A】`、`【买点C-B】`、`【买点D1回封】`、`【买点D2弱转强】`、`【买点D3低吸】`、`【买点B冰点】`
3. 卖点信号：`【卖出-主动止损】`、`【卖出-标准止损】`、`【卖出-强制清仓】`、`【卖出-退潮预警】`、`【卖出-退潮确认】`
4. 辅助信号：`【主线评分-核心/次级/轮动/放弃】`、`【角色识别】`、`【环境等级动态】`
