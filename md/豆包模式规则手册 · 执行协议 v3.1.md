# 豆包模式 · 执行协议 v3.1 投喂精华版

**修订说明**：本版本在 v3.0 基础上，依据“指数-主线-个股”统一框架，新增 **Ca日角色锚定规则（12.5）**、**指数共振确认（第四章）**、**确定性原则（系统内核前）** 及 **铁律10（固定锚，定操作）**，旨在将选股与指数共振日固定，实现每日操作的确定性输出。

---

## 系统内核与铁律（前置声明）

### 确定性原则
豆包模式是一套**确定性状态机**。只要输入数据完整、客观，每日的正确操作（状态、主线、角色、买点/卖点）在理论上是**唯一且可预先计算**的。实践中因“隔日确认”规则（铁律6）导致的延迟，不影响其最终唯一性。交易者应摒弃主观预判，完全依赖规则输出。

### 十**条铁律（绝对不可违背）**：
1. **锚点永不变**：预置锚点一旦确定，在满足三日缓冲条件前绝不更换。
2. **中军优先判主线，三日缓冲防误判**。
3. **候选池里多线程，启动企稳平等看**。
4. **退潮空仓不抄底，横盘混沌等风来**。
5. **连板断板走，趋势破线清**。
6. **止损不是破线等，弱信号来提前走**。
7. **G/D切换隔日定，当天信号不翻牌**。
8. **主线优先：主升期只做主线，放弃支线轮动**。
9. **执行清单只列当日可执行买点，观察日清单为空**。
10. **固定锚，定操作**：每日操作由当前市场状态唯一决定。状态 = f(锚点T的阶段, 环境等级, 主线评分)。输入当日客观数据，输出唯一的状态和操作清单。禁止在状态未明确时“抢跑”或“猜顶底”。

## 一、状态机（唯一判定依据）

**市场三种状态**：
```
正常主升周期：Ca → Cb → G1 → D1 → G2 →..D(N-1) → G(N)..D(N) → F → E
混沌期（H）：E后无主线，等新方向走出来
妖股模式（Y）：E后市场主跌，连板独立抱团
```

**状态机总览**：
```
Ca → Cb → G1 → D1 → G2 → D2──────────────────────────┐
                          │                          │
                          ▼                          │
              ┌──────────────────────┐               │
              │   GN-n → DN-n 循环   │               │
              │  (主升确立后的       │               │
              │   分歧-主升反复)     │ ───┐          │
              └──────────┬───────────┘    │          │
                         │                │          │
                         ▼                │          │
      某次DN-n触发破位 → F → 超预期 → 回到主升循环   │
                         │                           │
                   ┌─────┴─────┐                     │
                   ↓           ↓                     │
             有新方向启动  无方向+主跌               │
                   ↓           ↓                     │
                   H           Y                     │
                   ↓                                 │
           Ca → D1 → 尝试G2                          │
                   ↓                                 │
             ┌─────┴─────┐                           │
             ↓           ↓                           │
         成功→G2-1   失败→P→退回H→B冰点套利          │
             │                                       │
             └───────────────────────────────────────┘
```

### 状态速查表

| 状态 | 名称 | 定义 | 量化触发条件 | 操作指令 |
|------|------|------|-------------|---------|
| **Ca或者A** | 启动日 | 板块首次批量涨停 | ≥2只一字板(秒板)+≥1换手板+板块>2%；趋势型：中军放量>5%+板块成交创20日新高+板块>2% | 预置锚点，执行清单空 |
| **Cb** | 发酵确认 | Ca次日确认发酵延续 | 龙头封单较Ca增≥50%，且新增≥2首板 | 可加仓至仓2-3，执行买点C |
| **G1-n** | 首次主升 | 板块首次持续上涨 | 中军沿5日线上行或滞涨；n每日递增，跳过非交易日 | 持有不加仓 |
| **D1-n** | 首次分歧 | 主升后首次回调考验 | 盘中先锋炸板/中军回踩，次日确认；强分歧：先锋断板+炸板率>40%+中军破5 | D1-1：空仓等活口；D1-2：尾盘D3低吸(仓2-3)；D1-3：隔日竞价跟随(仓2-3)；D1+：减仓预警 |
| **G2-1** | **主升正式确立** | D1后弱转强修复成功，次日确认 | 跟风平均涨幅≥2%、中军站稳5日线、扩散系数≥0.8 | 持有不加仓。此为**主升确立点** |
| **G(N)-n** | 主升 | 确立主升后持续上涨 | 中军沿5日线上行或滞涨不创新高；n每日递增,G-D循环N次 | 持有不加仓 |
| **D(N)-n** | N次G后的分歧调整 | 确立主升后第N次 | 中军回踩关键均线，活口分离确认；n每日递增 | 活口分离确认买入，中军企稳回踩关键均线低吸 |
| **F** | **破位反弹（B浪修复）** | 主升后前一日出现破位信号，修复尝试 | 前一日已出现破位信号（中军破20日线 或 板块批量跌停≥3只）。今日出现修复尝试（跟风平均涨幅>0，但未站回关键均线） | 轻仓持龙头观察。修复超预期→G（N+1），切换下一阶段主升；分离信号触发→确认E退潮 |
| **P** | **炮灰**（修复失败） | **主升确立前**（H中），D1后尝试进入G2但失败 | 连续2日跟风均涨幅<2%+扩散系数<0.8+中军未站回5日线 | 减仓至仓1或清仓，退回H |
| **E** | 退潮确认 | 主线正式结束 | 中军破20日线+批量跌停≥3只+评分<3.0连续2日；快速通道：跌停>50家+主线中军跌停+指数跌>5%三者其二立即E | **清仓该板块所有持仓** |
| **H** | 混沌期 | E后无新主线，但市场非主跌 | E已触发，无新Ca确立G2-1 | **空仓等待**。有Ca则记录观察，能走完D1→G2则新周期开启；失败则P标记退回H |
| **B** | 冰点套利 | 混沌冰点后的短暂修复 | 连跌(上涨家数<1500)≥2天+前主线核心止跌缩量企稳 | 尾盘低吸(仓1)，隔日套利，失败退回H |
| **Y** | 妖股模式 | C级环境+市场主跌+无主线 | 指数<20日线+均线空头+连板≥5板且独立走强 | 选股≥3板，仓2，单票≤30%，断板即走 |

### 转移规则

- 主路径：Ca→Cb→G1→D1→G2→..D(N-1)→G(N)..D(N) → F → E, D和G后面的数字N代表一致(上涨)分歧(调整)次数,一定是**第N次的G在前D在后**.
- Ca可直接进G1（V型反转）,也可以日内启动到高潮隔日进入D1。状态可以跳过但必须符合客观节奏规律。
- 强制继承：按历史快照n(该状态持续交易日数，必须跳过非交易日)值递推，状态不变则n+1，状态变则重置为1
- 强势主升：D和G后面的数字N递推循环构成主升浪，强势主线可达5(N)浪即G1-D1-G2-D2-G3-D3...一直循环
- 锚点切换：须三日缓冲（挑战者连续3日评分≥旧锚点+中军连续3日站上5日线+旧锚点中军连续3日跌破10日线）；若未满足但候选处G2阶段可双锚点并行最长5天
- E后分支：有新方向→H；无方向+主跌→Y
- H中D1修复失败→P→退回H
- 全景表状态栏锚定在**最先启动、贯穿最久的主线锚点**的周期标记上。锚点对应状态就是市场状态，**市场状态唯一**，优先级锚点（T、H、Y）> 预置锚点 > 候选池支线。
- **主线分歧破位不要立即切换E，需要等F-E确认, F修复超预期进入下一个G（N+1）主路径**
- H与Y的状态确认时间：
1. **混沌期H**：E确认后的第一个交易日自动进入H，无需额外时间确认。H持续期间，每日检查是否出现新Ca方向（Ca定义见速查表）。若连续5个交易日无Ca且未进入Y，可启用B冰点套利，但主线锚点仍为空。
2. **妖股模式Y**：需**连续2个交易日**同时满足以下全部条件方可确认进入Y：
    - 沪指收盘价位于20日线下方，且5/10/20日均线呈空头排列；
    - 存在独立连板股（≥5板），且其所属板块涨停家数<3家；
    - 无任一板块主线评分≥4.0。
    若连续2个交易日任一条件不满足，则退出Y，回归H或等待新主线。
- 混沌期与妖股模式的锚点设置：
1. 混沌期起始锚点（H0）：E确认日的次日记为H0，状态标记为`H（混沌期，H+0）`。每日递推H+n，用于跟踪混沌期时长。H0不是主线锚点，不参与Ca→G2判定。
2. 妖股模式起始锚点（Y0）：Y模式确认日的首日记为Y0，状态标记为`Y（妖股模式，Y+0）`，并记录核心妖股代码（如“利仁科技 001259”）。每日递推Y+n，妖股断板或无接力则Y模式终止，回归H或新主线。

## 二、环境等级与仓位上限

| 等级 | 量化标准（三项） | 总仓位 |
|------|-----------------|--------|
| **S级** | 沪指>20日线 + 成交>2万亿 + 连板高度>5（缺一不可） | 仓4(40%) |
| **A级** | 沪指站上5日线 + 成交1.5-2万亿 | 仓3(30%) |
| **B级** | 沪指与5/20日线粘合，成交1-1.5万亿 | 仓2(20%) |
| **C级** | 沪指<20日线，成交<1万亿 | 仓1(10%) |

盘中动态升级：半日成交≥前日全日×0.8且沪指涨>1.5%；或上涨>4000家且涨停>80家，则当日自动上调一级。

## 三、主线强度评分（AI内部计算不需要展示步骤）

评分卡：

| 指标/权重 | 5分 | 3分 | 1分 |
|----------|-----|-----|-----|
| 政策强度(20%) | 国家级重磅 | 地方政策 | 无 |
| 资金流入(30%) | 板块涨幅>5% | >2% | <2% |
| 涨停数量(20%) | ≥15家 | ≥10家 | <5家 |
| 先锋封单(15%) | >5亿 | >2亿 | ≤2亿 |
| 中军表现(15%) | 核心中军涨停 | 核心中军涨>5% | 核心中军平或跌 |

阈值：≥4.5核心主线；4.0-4.4次级主线；3.0-3.9轮动方向；<3.0放弃, （注：该阈值可基于环境等级或市场风格相对下调，下调幅度及使用标准必须在复盘第3项中明确标注。）。

**执行流程（必须逐条，不可跳过）**：
1. 涨停数量：筛选板块涨停，炸板不算。
2. 先锋封单：取板块内最大封单额。
3. 资金流入：板块涨幅，辅以主力净流入微调（>10亿上调一档，<-10亿下调一档）。
4. 中军表现：取核心中军涨幅。
5. 政策强度：从产业催化判断。

## 四、锚点与候选池管理

- **锚点T**：
 1.锚点T对应主线退潮确认后预置锚点t晋级成正式锚点T。
 2.如果没有预置锚点，候选池方向通过三日缓冲(即三个交易日评分高于主线)取代。
 3.**锚点只可被取代不可取消，即使主线确认退潮**,如果没有主线则对应H和Y状态下的起点作为锚点取代（见状态机转移规则）。
 4.旧锚点T在E确认后不再作为交易参考，但保留其标记用于历史全景表。当前市场的“有效锚点”切换为H0或Y0，直到出现新主线Ca。
- **预置锚点t**：退潮窗口期（E后2交易日）内第一个Ca日方向，立即设为预置锚点，不可更改。
- **指数共振确认**：预置锚点t的Ca日必须与**指数企稳点**（沪指在连续下跌后首次收阳或长下影线，且上涨家数从冰点回升）同时发生，否则该方向仅作为普通候选，不设为预置锚点。若指数未企稳而板块独立走强，则按Y模式或轮动处理。
- **候选池**：窗口期内Ca→窗口期候选（优先）；窗口期外Ca→普通候选；连续2日评分<3.0且中军破20日线淘汰。
- **候选加速晋级**：非窗口期下，候选方向完成C→D1→G2-1且G2-1评分≥4.0、核心中军涨幅≥3%，即可直接进入双锚点并行期（旧主线仓2，新方向仓2）。
- **全景表状态节奏**：以主线启动点为锚点T，主线节奏为市场状态。其他方向维持自己的节奏状态。

## 五、三大核心角色买卖锚定

> 各角色的识别标准（量比、封单、连板数等）详见第十二章12.3节及新增12.5节。本章仅规定买卖锚定逻辑与止损规则。

| 角色 | 核心锚定 | 止损规则 |
|------|----------|----------|
| **核心中军** | 均线系统 | 破10日线减半，破20日线清仓 |
| **趋势先锋** | 新高（均线辅助） | 三日不新高或破5日线，减半或清仓 |
| **连板先锋** | 量能 + 断板 | 断板即走，不参与首阴、不反包 |

注：活口、卡位为临时观察角色（见12.3节），不设独立止损规则。

## 七、卖出与止损量化

### 7.0止损原则
| 层次 | 触发条件 | 执行 |
|------|---------|------|
| 主动止损 | 低开>3%无承接；冲高回落次日低开 | 开盘15分钟内 |
| 标准止损 | 破5日线减半；破20日线或断板清仓 | 收盘前 |
| 强制清仓 | 跌停；板块批量跌停≥3只；指数跌>3% | 盘中立即 |
| 退潮预警 | 中军连续2日破5；评分连续2日<4.0；先锋断板无回封 | 减仓 |
| 退潮确认E | 中军破20日线+批量跌停≥3只+评分<3.5 | 清仓 |

### 7.1止损规则差异
| 风格 | 先锋止损 | 中军止损 |
|------|----------|----------|
| **趋势主导** | 三日不新高或跌破5日线 | 跌破10日线减半，20日线清仓 |
| **连板主导** | 断板即走 | 跌破5日线减半 |

### 7.2 辅助离场信号（非强制）
| 信号 | 操作 | 说明 |
|------|------|------|
| 乖离率＞10%+滞涨 | 可减仓1/3 | 主动止盈 |
| 乖离率＞15%+滞涨 | 可再减1/3 | 主动止盈 |

## 九、每日复盘清单格式（9项）

每日收盘后，严格按照以下9项顺序填写，其中**第1项增加“环境等级”**，**第5项增加“环境等级”列**，**第8项标注弹性参数**。

| 序号 | 检查项 | 核心内容 |
|------|--------|----------|
| 1 | **大盘状态** | 指数表现、成交额、个股涨跌、20日线位置、当前模式、总仓位上限；阶段结论与资金逻辑；次日预判（支撑/压力、关键变量） |
| 2 | **情绪周期（全市场）** | 涨跌家数、涨停数据、连板高度、晋级率、核心特征；阶段结论；T日锚点和预置锚点t（最近一次T日、共振方向、当前T+N） |
| 3 | **主线评分与核心标的** | 按板块逐一列出评分表（含各指标权重得分及总分），注明判定、先锋龙头(连扳先锋和趋势先锋)、核心中军。展示排序：✅主线优先，⏳次之，❌最后 |
| 4 | **全景节奏表** | 以日期为行，指数状态和成交额、全市场情绪、主线阶段标记、操作要点。**必须包含指数、连板收益和主线阶段序列** |
| 5 | **主线动态跟踪表** | 方向、核心中军、阶段标记、风格（趋势还是连扳）、中军趋势（3日）、异动信号、角色、主线候选、次日弱转强标的、备注。排序同第3项 |
| 6 | **主线稳定性评估（含纵向跟踪）** | 当前主线、潜在挑战者、中军对比、3天缓冲、纵向阶段演变、切换结论；可选增加“龙头演化速览” |
| 7 | **明日计划** | 个股执行清单（标的+买点+执行时间+条件+仓位），按角色优先级排序；买点类型预案（当日/隔日分组）；核心提醒 |

| 9 | **主观输入(不输入为默认)** | 用户填写（周期锚点、主线、龙头、当前持仓等） |

## 十二、复盘报告生成补充协议（**此部分仅作为补充和案例，以第九章复盘清单为准，协议中没有说明的需要AI自行完善确保复盘报告全面客观**）

### 12.0 初始状态获取
1. 从 `【历史策略快照】` 提取昨日全景表**最后一行**的“状态”、“主线锚点及阶段”、“候选池及阶段”作为今日初始状态（AI内部处理不要展示）。
2. 若今日数据满足状态转移条件，更新状态；否则状态不变，天数(n)按交易日历递推增加1。
3. 若状态不变则**强制提取**主线锚点当前所处的阶段标记(如G2-3中的数字3)，作为今日n值计算的起点。

### 12.1 环境等级判定
1. 读取`【大盘指数数据】`的上证收盘点位、20日均线、成交额(沪+深)。
2. 读取`【涨停个股数据】`中的最高连板高度。
3. 按三项标准（指数>20日线，成交额>2万亿，连板高度>5）判定S/A/B/C。

### 12.2 主线评分流程
1. 从`【板块行情数据】`读取对应板块的涨幅、涨跌比、主力净流入。
2. 从`【涨停个股数据】`筛选该方向涨停股，统计数量。**归属以‘板块’字段为准**，一只股票最多归入一个方向。
3. 从`【核心中军行情数据】中筛选该方向的核心中军。核心中军的定义为：该方向中成交额最大且市值居前（若冲突，成交额优先）。取该核心中军的当日涨幅作为评分依据。若该方向无明确单一大市值中军（如板块分散），则取成交额前三名的算术平均涨幅。
4. 从`【产业催化电报】` 获取政策/产业消息计算政策强度评分。
5. 按五项权重计算得分。**评分过程不出现于正文。**

### 12.3 角色输出规范（常规日）
1. 主线方向，尽量输出以下所有角色清单。清单末尾注明“**当前最优标的：[标的名称]，理由：[基于风格优先级和数据的简短理由]**”。
2. **只做主线，不做支线轮动**，支线角色只观察不买入必须标明。

| 角色 | 识别标准 |
|------|---------|
| **连板先锋** | 属该方向，**连板数最高且至少大于等于2**，封板时间早，封单强。只能10cm主板。*注：在Ca日（指数共振日），连板先锋的识别应遵循12.5.1的锚定规则（时间窗口放宽至前1日）。非Ca日则按常规标准（仅当日）识别。* |
| **趋势先锋** | 属该方向，量比≥1.5，趋势评分最高，≤2板。优先100元以内10cm主板。*注：在Ca日（指数共振日），趋势先锋的识别应遵循12.5.2的锚定规则。* |
| **核心中军** | 属该方向，核心中军的定义见12.2节3条。*注：在Ca日（指数共振日），核心中军的识别应遵循12.5.3的锚定规则。* |
| **活口** | 属该方向，分歧日跌幅<3%，分时在均线上方。 |
| **卡位** | 龙头断板当日逆势涨停。 |

主线风格判断：计算（连板先锋数量 + 3×涨停股中连板占比）与（核心中军涨幅% + 5×核心中军成交额占板块比例），若前者大于后者则为连板主导，否则为趋势主导。
- **趋势主导风格**下，优先趋势先锋 > 连板先锋 > 核心中军。
- **连板主导风格**下，优先连板先锋 > 趋势先锋 > 核心中军。

### 12.4 操作建议生成
1. 从全景表确定当前**状态标记**。
2. 从**状态速查表**检索对应**操作指令**。
3. 将操作指令与角色清单中的最有代表的标的填入全景表。
4. 将操作指令、买卖点规则与角色清单中的最优标的结合，填入明日计划（复盘报告）或者执行清单（盘前策略）。

### 12.5 Ca日角色锚定规则（指数共振日的标准化识别）**【新增】**
在指数企稳点（连续下跌后首次止跌或长下影线）出现共振的Ca日（首次批量涨停），应在此固定窗口内完成主线角色的初始锚定。该锚定结果贯穿后续整个主升周期，除非角色被更强者取代。

#### 12.5.1 连板先锋锚定
- **时间窗口**：Ca日及**前1个交易日**。
- **识别标准**：取该时间窗口内，属于该方向且**连板数最高（至少≥2板）** 的10cm主板标的。若最高连板数=1（即只有首板），则**当日无连板先锋**，以趋势先锋为主。
- **原因**：避免漏掉提前1日启动的龙头（如电力5月18日启动，5月19日指数共振日已4板），同时防止将普通首板误认为连板先锋。

#### 12.5.2 趋势先锋锚定
- **时间窗口**：Ca日当天。
- **识别标准**：取**量比≥1.5** 且 **涨幅≥7%（10cm主板）或≥10%（20cm）** 的标的，按量比从高到低排序取第一。若无满足标的，则取当日**成交额最大的涨停板**（作为趋势中军替代先锋角色）。
- **原因**：量比≥1.5是主动进攻的量化门槛，涨幅要求过滤跟风杂毛。

#### 12.5.3 核心中军锚定
- **时间窗口**：Ca日当天。
- **识别标准**：取**成交额最大**且**市值居前（板块内前3）** 的标的。若该标的同时满足趋势先锋锚定条件，则优先标记为趋势先锋，核心中军另选成交额第二大的标的。
- **原因**：完全遵循规则手册原定义（12.2节3条），不添加主观条件。

#### 12.5.4 后续确认与调整
- 锚定后的角色在后续G/D循环中可能被新的先锋超越（如新连板龙头出现），此时应**动态替换**：新连板先锋的连板数必须大于旧先锋，且封单更强。
- 趋势先锋若被卡位（量比更高、趋势评分更高），则替换。
- 核心中军一般不变（除非出现明显的成交额碾压）。

### 12.6 全景表输出格式
1. 所有复盘报告必须包含此表，且作为下一日输入载体。
2. 全景表必须从T（锚点日）开始，中途可以跳过。
3. 状态栏是状态机中的一种状态标记，状态唯一并且与锚点、主线状态一致，状态转变必须符合状态机变化的客观规律和转移规则。

### 12.7 节奏模型 （**节奏模型是主线的“生理年龄”，状态机才是法律，两者冲突时绝对服从状态机。**）

| 环境 | 周期长度 | 节奏数组 | 数组节点含义 |
| :--- | :--- | :--- | :--- |
| **S级** | 13日周期 | [**0**, 2, 4, 9, 11] | **0=启动**，2=分歧，4=确立，9=衰竭，11=退潮 |
| **A/B级** | 9日周期 | [**0**, 2, 4, 6, 8] | **0=启动**，2=分歧，4=确立，6=衰竭，8=退潮 |
| **C级** | 无固定节奏 | null | 仅适用妖股模式 |
