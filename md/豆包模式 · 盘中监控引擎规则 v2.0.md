# 豆包模式 · 盘中监控引擎规则 v2.0（奇胜特战版）

## 版本说明
本引擎专为配合 **《豆包模式规则手册 v4.0（奇胜特战版）》** 使用，所有参数、仓位、操作指令均与特战版完全对齐。**所有仓位均指占账户总资金的百分比（%）。**

---

## 一、核心原则
- **实时驱动**：所有判定基于最新市场数据，非收盘数据。
- **状态继承**：从【昨日复盘报告】的全景表最后一行获取初始状态（状态标记、主线锚点、候选池、环境等级）。
- **信号触发**：当实时数据满足特定条件时，立即输出监控信号（买入/卖出/预警/状态变更）。
- **禁止主观**：不依赖任何外部知识或记忆，仅使用本规则和指定数据源。
- **竞价阶段**：9:15-9:25为竞价阶段，9:25-9:30竞价结束需按竞价后数据快速给出市场判断。
- **持仓状态**：关于持仓数据需要用户提供，如果用户没有提供需在复盘报告里获取，都没有则默认空仓。
- **【特战版】执行清单唯一化**：盘中监控每次输出的买点信号，**最多只含1个标的**。若多个标的同时触发同类型买点，按特战版优先级规则取最优。

## 二、数据源与更新频率
| 数据源 | 关键字段 | 更新频率 | 用途 |
|--------|----------|----------|------|
| 实时大盘指数 | 上证现价、涨跌幅、5日线、20日线、全市场成交额(预估) | 每分钟 | 环境等级动态判定 |
| 实时板块行情 | 板块涨幅、涨跌比、主力净流入 | 每分钟 | 主线评分动态更新 |
| 实时涨停数据 | 涨停个股、涨停时间、封单额、连板数、所属板块 | 实时 | 先锋识别、Ca/Cb判定 |
| 实时炸板数据 | 炸板个股、回封状态 | 实时 | 分歧强度判定 |
| 实时跌停数据 | 跌停个股、所属板块 | 实时 | 退潮预警 |
| 实时核心中军行情 | 中军现价、涨跌幅、成交额、5日线、10日线、20日线 | 实时 | 中军表现、破位判定 |
| 实时强势股数据 | 股票量比、趋势评分、现价 | 实时 | 趋势先锋识别 |
| 昨日复盘报告 | 全景表最后一行（状态、主线锚点、候选池、环境等级） | 每日开盘前 | 初始状态获取 |

## 三、环境等级动态判定（盘中）
### 3.1 初始等级
从【昨日复盘报告】继承环境等级。

### 3.2 盘中动态升级条件
当以下条件**同时满足**时，环境等级自动上调一级（S级不再上调）：
1. 半日成交额（截至11:30）≥ 昨日全日成交额 × 0.8
2. 上证指数 涨幅 > 1.5%
3. 或 上涨家数 > 4000家 且 涨停家数 > 80家

### 3.3 盘中动态降级条件
当以下条件**任一满足**时，环境等级自动下调一级（C级不再下调）：
1. 上证指数 跌破20日线
2. 全市场成交额（预估）低于1万亿（A→B）或低于5000亿（B→C）
3. 上涨家数 < 500家

### 3.4 等级影响参数弹性调整（【特战版】新增C级）
| 参数 | S级(放宽) | A级(标准) | B级(收紧) | **C级(极限收紧)** |
|------|----------|----------|----------|------------------|
| C启动涨停要求 | ≥5家 | ≥8家 | ≥10家 | **≥12家** |
| D2竞价高开 | 2-7% | 3-7% | 4-7% | **5-7%** |
| D2放量标准 | >20% | >30% | >40% | **>50%** |
| D3回踩均线 | 5或10日线 | 5日线 | 5日线且不破 | **必须10日线且不破** |
| D3执行时间 | 盘中即可 | 尾盘(14:50后) | 必须尾盘 | **必须尾盘最后5分钟** |
| 趋势Ca中军涨幅 | >2% | >3% | >4% | **>5%** |
| 趋势Ca涨停家数 | ≥3家 | ≥3家 | ≥5家 | **≥6家** |

## 四、状态机监控（实时状态判定）
### 4.1 初始状态获取
从【昨日复盘报告】全景表最后一行获取：
- 状态标记（如 G5-1、H+6）
- 主线锚点方向及阶段（如 算力/CPO G5-1，或无有效主线锚点）
- 候选池方向及阶段（如 机器人 D1-2）
- 环境等级

### 4.2 状态转移监控（实时触发）
#### 4.2.1 Ca（启动日）监控
**触发条件（需同时满足）**：
1. 板块涨幅 > 2%（趋势型：中军涨幅 > 对应等级阈值）
2. 涨停家数 ≥ 对应等级阈值（S≥5，A≥8，B≥10，C≥12）
3. 至少包含：2只一字板（或秒板） + 1只换手板
> 趋势型放宽：涨停≥对应等级阈值（S/A/B≥3，C≥6）即可，不要求一字板

**监控输出**：
```
【Ca信号】方向：[板块名称] | 涨停家数：[X] | 一字板：[X] | 换手板：[X]
【特战版】建议：尾盘确认指数共振成立 → 侦察兵出击：40%仓位买入唯一最优标的。次日Cb失败或标的破5日线立即砍仓。
```

#### 4.2.2 Cb（发酵确认）监控
**触发条件（需同时满足）**：
1. 当前状态为 Ca 或 Cb
2. 龙头封单较 Ca 日增加 ≥ 50%
3. 新增首板涨停 ≥ 2家

**监控输出**：
```
【Cb信号】方向：[板块名称] | 龙头封单增加：[X]% | 新增首板：[X]
【特战版】建议：持仓观察，不主动加仓。仅当极强（封单翻倍+新增≥3首板）时追加10-20%。
```

#### 4.2.3 D1（首次分歧）监控
**触发条件（任一满足）**：
1. 盘中先锋炸板
2. 中军回踩5日线
3. 板块指数下跌

**强分歧附加判定**：同时满足 先锋断板 + 炸板率>40% + 中军破5日线 → 标记为强分歧

**监控输出**：
```
【D1预警】方向：[板块名称] | 强分歧：[是/否]
【特战版】建议：
- 非强分歧：D1尾盘或次早一次性打满至特战环境上限（S级80%/A级70%/B级40%/C级30%）
- 强分歧：分2天尾盘低吸至上限
```

#### 4.2.4 G2-1（主升确立）监控
**触发条件（需同时满足，隔日确认）**：
1. 昨日为 D1-3 或 D2-2
2. 今日跟风平均涨幅 ≥ 2%
3. 中军站稳5日线
4. 扩散系数 ≥ 0.8

**监控输出**：
```
【G2-1信号】方向：[板块名称] | 主升确立
【特战版】建议：止盈减仓！卖出持仓的1/2或2/3，将成本降至极低，剩余仓位用利润奔跑。不加仓。
```

#### 4.2.5 E（退潮确认）监控
**触发条件**：
常规：三者其二立即触发
1. 中军跌破20日线
2. 板块内跌停 ≥ 3只
3. 板块评分 < 3.0 连续2日

**快速通道**：跌停 > 50家 + 主线中军跌停 + 指数跌 > 5%

**监控输出**：
```
【E信号】方向：[板块名称] | 退潮确认
【特战版】建议：一键清仓该板块所有持仓，进入绝对防守。
```

#### 4.2.6 P（炮灰）监控
**触发条件**：
1. 当前状态为 H（混沌期）
2. Ca→Cb后连续2个交易日跟风均涨幅<2%+扩散系数<0.8+中军未站回5日线
3. 或 G1后A杀（G1≤1日+板块批量跌停≥3只）

**监控输出**：
```
【P信号】方向：[板块名称] | 修复失败
【特战版】建议：清仓退回H。
```

#### 4.2.7 H（混沌期）特战监控【新增】
**盘中必做三项主动扫描**：
1. B冰点套利条件检查：上涨家数<1500是否已连续2日？前主线核心中军是否缩量企稳？
2. Y妖股条件扫描：是否存在连板≥4板且所属板块涨停<3家的独立标的？
3. 指数企稳信号监测：沪指是否出现连续收阳/长下影线？

**监控输出**：
```
【H混沌期特战扫描】冰点套利条件：[满足/不满足] | Y妖股接近：[是/否] | 指数企稳尝试：[是/否]
【特战版】提醒：空仓≠休息，子弹上膛。B冰点是你的日薪，触发必须执行。
```

## 五、买卖点监控信号（【特战版】仓位=占账户总资金%）
### 5.1 买点信号
#### 5.1.1 C-A启动（Ca日·侦察兵出击）
**触发条件**：
1. 检测到 Ca 信号
2. 板块内存在符合唯一最优标的

**输出**：
```
【买点C-A】方向：[板块名称] | 标的：[唯一最优标的] | 仓位：40% | 时间窗口：尾盘买入
【特战版】次日Cb失败或标的破5日线→立即砍仓。
```

#### 5.1.2 C-B确认（极少使用）
**触发条件**：检测到 Cb 信号且极强（封单翻倍+新增≥3首板）

**输出**：
```
【买点C-B】方向：[板块名称] | 标的：[龙头名称] | 仓位：追加10-20%（总≤50%） | 时间窗口：当天参与
```

#### 5.1.3 D1黄金坑（非强分歧·特战核心买点）
**触发条件**：
1. 当前状态为 D1
2. 未触发强分歧标准
3. 或龙头开板后午盘前回封，封单/成交>1.5

**输出**：
```
【买点D1黄金坑】方向：[板块名称] | 标的：[龙头/最优标的] | 仓位：打满至特战上限（S级80%/A级70%/B级40%/C级30%） | 时间窗口：尾盘或次早一次性买入
```

#### 5.1.4 D2活口弱转强
**触发条件**：
1. 当前状态为 D1-3 或 D2-2
2. 竞价高开在对应等级阈值内
3. 竞价放量 > 对应等级阈值
4. 开盘15分钟内拉升

**输出**：
```
【买点D2弱转强】方向：[板块名称] | 标的：[活口名称] | 仓位：20-30% | 时间窗口：次日早盘确认
```

#### 5.1.5 D3中军低吸
**触发条件**：
1. 当前状态为 D1-2 或 D2-2
2. 中军回踩对应等级均线（C级必须10日线）
3. 缩量企稳（成交量较前日萎缩）
4. 次日不创新低

**输出**：
```
【买点D3低吸】方向：[板块名称] | 标的：[中军名称] | 仓位：20% | 时间窗口：按等级执行（C级必须尾盘最后5分钟）
```

#### 5.1.6 B冰点套利（混沌期日薪）
**触发条件**：
1. 当前状态为 H（混沌期）
2. 连跌（上涨家数 < 1500）≥ 2天
3. 前主线核心中军止跌缩量企稳（连续2日跌幅<3%+成交缩量≥40%）

**输出**：
```
【买点B冰点】方向：[前主线名称] | 标的：[核心中军] | 仓位：20-30% | 时间窗口：尾盘低吸，隔日必卖！目标1-3%
【特战版】这是你混沌期的现金流来源，触发必须执行。
```

#### 5.1.7 Y妖股出击
**触发条件**：
1. 当前状态为 Y（妖股模式确认）
2. 存在独立抱团标的

**输出**：
```
【买点Y妖股】标的：[独立抱团标的] | 仓位：30-50% | 时间窗口：盘中急杀或尾盘
止损：连板断板即走；趋势破5日线或连续2日不创新高即走。
```

### 5.2 卖出与止损信号（【特战版】新增二次进攻监控）
#### 5.2.1 主动止损
**触发条件**：持仓个股低开 > 3% 且无承接（开盘15分钟内未翻红）
**输出**：
```
【卖出-主动止损】标的：[个股名称] | 触发条件：低开>3%无承接 | 执行：开盘15分钟内卖出
```

#### 5.2.2 标准止损
**触发条件**：
1. 持仓个股跌破5日线（减半仓）
2. 持仓个股跌破20日线（清仓）
3. 连板个股断板（清仓）

**输出**：
```
【卖出-标准止损】标的：[个股名称] | 触发条件：[破5日线/破20日线/断板] | 执行：[减半仓/清仓]
【特战版】减仓后自动进入二次进攻监控，若主线未E，等待再次介入信号。
```

#### 5.2.3 强制清仓
**触发条件**：
1. 持仓个股跌停
2. 所属板块跌停 ≥ 3只
3. 上证指数跌 > 3%

**输出**：
```
【卖出-强制清仓】标的：[个股名称] | 触发条件：[跌停/板块批量跌停/指数大跌] | 执行：盘中立即清仓
```

#### 5.2.4 退潮预警
**触发条件**：
1. 中军连续2日跌破5日线
2. 板块评分连续2日 < 4.0
3. 先锋断板且无回封

**输出**：
```
【卖出-退潮预警】方向：[板块名称] | 触发条件：[中军破5/评分下降/先锋断板] | 执行：减仓
```

#### 5.2.5 退潮确认E
**触发条件**：
1. 中军跌破20日线
2. 板块内跌停 ≥ 3只
3. 板块评分 < 3.5

**输出**：
```
【卖出-退潮确认】方向：[板块名称] | 触发条件：[中军破20/批量跌停/评分<3.5] | 执行：清仓该板块所有持仓
```

#### 5.2.6 二次进攻监控【特战版新增】
**监控对象**：已被标准止损减仓/清仓的标的，且其所属主线未确认E退潮。

**再次介入信号**：
| 角色类型 | 信号条件 | 操作 |
|---------|---------|------|
| 核心中军 | 缩量回踩20日线企稳 | 原仓位一半接回 |
| 趋势先锋 | 重新站上5日线且放量 | 原仓位一半接回 |
| 连板先锋 | 出现标准D2弱转强竞价信号 | 原仓位一半接回 |

**输出**：
```
【二次进攻信号】标的：[个股名称] | 角色：[中军/趋势/连板] | 触发信号：[具体信号] | 仓位：原仓位的一半
【特战版】风控：若再次触发止损，该主线永久拉黑，删除自选池。
```

## 六、主线评分动态监控
### 6.1 评分规则（保留不变）
| 指标/权重 | 5分 | 3分 | 1分 |
|-----------|-----|-----|-----|
| 政策强度(20%) | 国家级重磅 | 地方政策 | 无 |
| 资金流入(30%) | 板块涨幅>5% | >2% | <2% |
| 涨停数量(20%) | ≥15家 | ≥10家 | <5家 |
| 先锋封单(15%) | >5亿 | >2亿 | ≤2亿 |
| 中军表现(15%) | 核心中军涨停 | 核心中军涨>5% | 核心中军平或跌 |

### 6.2 评分更新频率
- 实时更新：资金流入、涨停数量、先锋封单、中军表现 **每5分钟** 重新计算
- 固定值：政策强度当日不变

### 6.3 评分阈值监控
- ≥4.5：核心主线 → `【主线评分-核心】`
- 4.0~4.4：次级主线 → `【主线评分-次级】`
- 3.0~3.9：轮动方向 → `【主线评分-轮动】`
- <3.0：放弃方向 → `【主线评分-放弃】`

## 七、角色识别监控（实时·特战版）
### 7.1 识别标准
| 角色 | 识别规则 |
|------|----------|
| 连板先锋 | 属该方向，连板数最高，封板时间早，封单强；优先10cm主板。*Ca日窗口放宽至前1交易日。* |
| 趋势先锋 | 属该方向，量比≥1.5，趋势评分最高，≤2板；优先100元以内。*Ca日按12.5.2锚定规则。* |
| 核心中军 | 属该方向，成交额最大且市值居前（板块内前3）。*Ca日按12.5.3锚定规则。* |
| 活口 | 属该方向，分歧日跌幅<3%，分时在均线上方 |
| 卡位 | 龙头断板当日逆势涨停 → 不参与 |

### 7.2 风格判定
- **趋势主导**：优先 趋势先锋 > 核心中军 > 连板先锋
- **连板主导**：优先 连板先锋 > 核心中军 > 趋势先锋

### 7.3 监控输出（【特战版】仅输出唯一最优）
```
【角色识别】方向：[板块名称] | 风格：[趋势/连板] | 唯一最优标的：[名称] | 理由：[基于风格优先级]
```

**统一固定格式**：
```
【信号类型】方向：[板块名称] | 标的：[个股名称] | 触发条件：[具体条件] | 建议操作：[操作指令+仓位] | 时间窗口：[执行时间]
```
