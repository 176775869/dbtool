# 豆包模式执行附录 —— 双版本

根据你的需求，我提供两个版本：

- **版本A：标准版（v3.8.12-exec）**：严格遵循豆包模式核心规则，低频、高确定性，适合实盘稳定盈利。
- **版本B：学习版（v3.8.15-learn-exec）**：适当放宽限制，增加交易频率，供模拟盘学习练手，帮助新手积累经验。

两个版本独立使用，**学习版仅限模拟盘**，不可用于实盘。

---

## 版本A：标准版（v3.8.12-exec）

> 完整内容同你已提供的 v3.8.12-exec，此处仅列出核心差异点，完整版见前文。

**核心特征**：
- 只做主线，主线确认后支线买点一律放弃。
- 买点频率：平均每周1-2次。
- 仓位严格：仓1-仓4按百分比。
- 观察日（1D-1/2D-1）执行清单为空。
- 分歧至少2天才能弱转强。
- 适用：实盘、追求稳定复利。

---

## 版本B：学习版（v3.8.15-learn-exec）

> **版本**：v3.8.15-learn-exec（模拟盘学习专用）
> **说明**：本版本在豆包模式框架基础上，**为模拟盘学习目的**适当放宽限制，增加交易频率，帮助新手快速积累盘感、熟悉买点。**严禁用于实盘**。学习版不改变主线识别、评分卡、仓位上限等核心风控，仅在“参与范围”和“分歧观察期”做松绑。

### 📋 一、学习版与标准版的核心差异

| 项目 | 标准版（v3.8.12-exec） | 学习版（v3.8.15-learn-exec） |
|------|------------------------|------------------------------|
| **主线确认后支线** | 禁止参与 | **允许参与**，但仓位≤仓1（10%），且需满足支线买点条件 |
| **分歧观察日（1D-1）** | 执行清单为空，不买入 | **允许尾盘以仓0.5（5%）试错活口**（弱分歧条件同标准版） |
| **分歧转强天数** | 至少2天分歧才能弱转强 | **允许1天分歧后次日直接试错**（但需满足弱转强量化条件） |
| **冰点套利（买点B）** | 仅限前主线龙头 | **可扩展至当日最强支线龙头**（需评分≥4.0） |
| **持仓时间** | 主升期持有，断板/破位走 | **可主动止盈**（如连板断板前减半，不等断板） |
| **仓位上限** | 仓4（40%） | 保持不变，但支线仓位从主线额度中扣除 |
| **单日最大交易次数** | 无限制但实际很少 | **建议≤3笔**（模拟盘学习） |

### 📌 三、学习版关键规则量化（松绑条款）

#### 3.1 支线参与规则（新增）

| 条件 | 要求 |
|------|------|
| 支线评分 | ≥4.0分（次级主线或候选） |
| 买点结构 | 出现买点C（启动或确认）或买点D（分歧活口） |
| 仓位 | **仓1（10%）**，且从总仓位中扣除 |
| 止损 | -5%或破5日线 |
| 持仓时间 | 最长2天，第3天强制离场 |
| 与主线冲突 | 主线出现买点时，支线仓位立即清仓 |

#### 3.2 分歧观察日允许试错（原1D-1/2D-1）

- **标准版**：不买入，只筛选活口。
- **学习版**：若满足“弱分歧”条件（板块跌幅≤1.5%、仍有≥3只涨停、中军跌幅≤2%），允许尾盘以**仓0.5（5%）** 试错活口（中军或先锋）。
- 次日若弱转强，可加仓至仓1-仓2；若走弱，开盘止损。

#### 3.3 分歧转强天数缩短

- **标准版**：趋势主线必须经历1D-2才能进入2G-1。
- **学习版**：允许1D-1尾盘试错后，次日若先锋竞价抢筹+开盘快速拉升，可直接视为2G-1，执行D2买点（仓位仓1-仓2）。但需做好严格止损（跌破分离日最低价）。

#### 3.4 冰点套利扩展

- **标准版**：买点B仅限前主线龙头。
- **学习版**：若全市场连续2天冰点（下跌>3000家），可参与**当日最强支线龙头**（评分≥4.0），仓位仓1，隔日套利。

#### 3.5 主动止盈规则（学习版可放宽）

| 持仓类型 | 标准版 | 学习版 |
|----------|--------|--------|
| 连板先锋 | 断板即走 | 可提前在断板前减半（如封单减弱、板上放量），剩余仓位断板走 |
| 趋势中军 | 破5日线减半，20日线清仓 | 乖离率>15%时可主动减仓1/3，锁定利润 |

### ⚠️ 五、学习版重要提示

1. **仅限模拟盘使用**。学习版放开了风控，实盘中容易导致重大回撤。
2. **每笔交易必须记录**：买点类型、是否符合条件、盈亏、是否遵守离场纪律。
3. **月度复盘统计**：对比标准版模拟盘和学习版的胜率、盈亏比、最大回撤，判断自己更适合哪种节奏。
4. **从学习版过渡到标准版**：当你在学习版中能稳定盈利（连续3个月正收益，回撤<10%），且发现自己大部分利润来自主线而非支线时，应切换到标准版实盘。
5. **学习版的目标不是赚钱，而是练手**：目的是熟悉买点、培养盘感、理解情绪周期。不要沉迷于高频交易。
