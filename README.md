## 💎 系统核心思想

豆包模式是一套以**主线识别为核心**、**情绪周期为节奏**、**量化标准为决策**、**严格纪律为保障**的短线龙头交易系统。通过客观量化过滤主观情绪，在主升期重仓出击，退潮期轻仓防守，实现长期稳定复利。
